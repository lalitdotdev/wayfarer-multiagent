# Security Policy

## Supported Versions

We follow semantic versioning. The following versions are currently being
supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Please report suspected security vulnerabilities via
[email to the project maintainers](mailto:lalit@example.com). You will receive
a response within 48 hours. If the issue is confirmed, we will release a patch
as soon as possible depending on complexity but historically within a few days.

Please do not disclose security-related issues publicly until a fix has been
announced.

We follow the principles of responsible disclosure.

## Preferred Languages

We prefer all communications to be in English.
